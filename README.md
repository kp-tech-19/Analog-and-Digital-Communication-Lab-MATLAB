# Analog-and-Digital-Communication-Lab-MATLAB

This repository contains MATLAB implementations and simulations for Analog and Digital Communication techniques.Each experiment includes MATLAB scripts, output plots, and interpretation of results.

## Experiments

1. Digital Representation of Analog Signals – PCM and DM
2. Generation and Coherent Detection of BPSK
3. Generation and Coherent Detection of BFSK
4. Modulated Signal Generation – DPSK, QPSK, and QAM Schemes  
5. Simulation of Constellation Diagrams – BPSK, QPSK, and QAM  
6. Implementation of (n, k) Error Control Coding – Linear Block Codes and Cyclic Codes  
7. Implementation of (n, k, K) Convolutional Coder
8. BER Analysis of M-ary PSK Communication Link
9. Simulation of Spread Spectrum Modulation Techniques

## Getting Started

1. Open MATLAB  
2. Clone or download this repository  
3. Navigate to any experiment folder  
4. Run the corresponding `.m` file  
5. Output plots are stored as `.png` inside each folder  

## Requirements

- MATLAB
